# [monin]

This is a simple Web3 project related to cryptocurrency and blockchain development.

## Features
- Smart Contract integration
- Web3.js / Ethers.js compatibility
- Built for EVM chains (Ethereum, Base, Polygon)

## Tech Stack
- Solidity
- JavaScript
- Hardhat/Foundry

## How to use
1. Clone the repository
2. Run `npm install`
3. Run `npm start`# Monin
